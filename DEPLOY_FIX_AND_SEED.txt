═══════════════════════════════════════════════════════════════════
WHY THIS FIX WORKS
═══════════════════════════════════════════════════════════════════

Serverless Problem:
- Each Lambda invocation might recreate Mongoose models
- mongoose.model() throws error if model already exists
- This caused "_id" errors

Solution:
- Check mongoose.models.ModelName first
- Only create if doesn't exist
- Works perfectly in serverless
